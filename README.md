# Dev Ratna Saxena — QA Portfolio

A modern, single-page portfolio for a Software QA Engineer (Manual + Accessibility Tester). Built with React (Vite), Tailwind CSS, and Lucide icons.

## Setup

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build

```bash
npm run build
npm run preview
```

## Resume

Place your resume PDF in `public/resume.pdf` so the "Download Resume" button works. You can copy `Myresume.pdf` to `public/resume.pdf`.

## Customize

- **Email**: Edit `src/components/Contact.jsx` and replace `devratna@example.com` with your email.
- **LinkedIn**: The LinkedIn URL is set to `https://www.linkedin.com/in/dev-ratna-saxena` in Hero and Contact; update if needed.
